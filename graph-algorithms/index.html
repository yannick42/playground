Graph algorithms
Hierholzer's algorithm (1873) to find an Eulerian path over a complete graph \(K_n\) (with an odd number \(n\))

Kruskal's algorithm (1956) to find the minimum spanning tree of a graph, using an union-find datastructure (based on linked lists)

A simple force-directed layout algorithm with "springs" and "electrons" (Eades, 1984) is used to have a nicer (less random) positioning of the vertices
