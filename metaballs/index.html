Metaballs
  • Based on this Youtube video by Reducible
  • Invented in 1982 by James Blinn (1949-...) for a TV documentary (to model atoms/DNA, in 3D)
  • Can be used with Marching squares/cubes algorithm (originally, 3D ray tracing?), it's a sum of implicit functions of multiple circles (or eventually, surfaces \(f(x,y,z)=0\)), the curve represents where this function crosses 0. → \(\sum\limits_i \text{metaball}_i = \text{threshold}\)
  • The curve is drawn using linear interpolation of this function over a grid
  • https://en.wikipedia.org/wiki/Metaballs
