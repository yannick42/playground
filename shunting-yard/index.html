Goal: a simple method to evaluate arithmetic (or logical) expressions, with parenthesis

Invented by Edsger Dijkstra in 1961. Using a stack, this operator-precedence parser produces a postfix (RPN) notation optimized for evaluation. Parsing input formula from left to right, and mainting an "op-stack", it decides where to route elements based on operators precedence order.
