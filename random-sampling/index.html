Random sampling
Methods: uniform, Poisson disk sampling, acceptance-rejection method
  • Poisson disk sampling allows to maintain an user-defined minimal spacing between the random points
  • The acceptance-rejection algorithm uses an uniform distribution (JS Math.random()) as the proposal distribution, and use the PDF (density function) of the Gaussian (µ=0, σ=1) as the desired/target distribution (centered on the canvas), but it can be any distribution. (It needs two values x and y).
  • Box-Muller transform (1958): by picking 2 random values from a uniform distribution.
